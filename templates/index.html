Version Checker

API Endpoint

GET: http://{{ host }}:{{ port }}{{ api_path }}?{{ query_params|safe }}


Examples

Request:  GET: http://{{ host }}:{{ port }}{{ api_path }}?ver_1=1.0.0&ver_2=1.0.1
Response:  {result: "1.0.0 is before 1.0.1"}


Request:  GET: http://{{ host }}:{{ port }}{{ api_path }}?ver_1=2.0&ver_2=1.0.1
Response:  {"result": "2.0 is after 1.0.1"}


Request:  GET: http://{{ host }}:{{ port }}{{ api_path }}?ver_1=2.0&ver_2=2.0
Response:  {"result": "2.0 is equal to 2.0"}
